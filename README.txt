Running `make` builds the compiler at `myc--`.

Example programs are found in the `rd` directory and
their corresponding example executions are found in the
`rd-execution` directory.

For each program, the following output is generated:
  - A mapping of numbered labels to statements
  - The list of edges between labels in the CFG
  - Printed entry and exit equations for RD analysis
  - Solutions to the RD analysis equations
